# erp_finaptico
ERP financiero desarrollado para Finaptico. Arquitectura basada en Odoo → n8n → Supabase → Lovable. El sistema centraliza datos contables y bancarios desde múltiples instancias de Odoo, los normaliza en Supabase y expone una capa de visualización para KPIs, previsiones fiscales e informes operativos para cada empresa cliente.
